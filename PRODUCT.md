# НЕЙРО — интерактивный курс по ИИ для подростков

**Register:** brand (дизайн — часть продукта; вовлечение и «вау» важны).

## Что это
Геймифицированный сайт-тренажёр к школьному курсу «ИИ с нуля» для подростков 12–17 лет.
Учитель ведёт занятие, сайт даёт интерактивные задания, квесты, XP и проверку.
Финальная цель курса (7 уроков) — каждый ученик запускает свою локальную LLM на ноуте.

## Аудитория
- Ученики 12–17 лет, без опыта программирования. Главные критерии: «трогать руками», мгновенная обратная связь, ощущение игры.
- Учитель — отдельный режим под паролем (методичка: что говорить, тайминг, ключи ответов).

## Дизайн-направление: «Нейро-Лаб»
Тёмная голографическая консоль. Живая нейросеть на canvas-фоне + аврора + плёночное зерно.
Неон по делу (аква/фиолет/розовый/золото), рёберная подсветка поверхностей, объёмная карта-квест.

- **Шрифты:** Unbounded (дисплей), Manrope (UI/текст), JetBrains Mono (числа HUD, мало).
- **Запреты (соблюдены):** без gradient-text, без тотального glass, без капс-трекинг-эйброу на каждой секции.
- **Геймификация:** XP, уровни, серии/стрик, маскот-робот Нейро, бейджи, звук (Web Audio, офлайн).

## Технически
Статический офлайн-сайт без сборки: `index.html` + `styles.css` + `app.js` (движок) + `data.js`
(контент и методичка) + `sound.js`. Прогресс в localStorage. Пароль учителя — вверху `app.js`.
Типы заданий: sort, axis, order, match, binary(стрик), hotspot, tokens, nextword, feed, slider.
Структура рассчитана на добавление уроков 2–7 правкой `data.js`.
