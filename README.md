# НЕЙРО · Урок 1 — «Что такое ИИ»

Интерактивный урок по ИИ для подростков 12–17 лет. Статический сайт без сборки —
открывается двойным кликом по `index.html` или публикуется на GitHub Pages.

- **7 миссий · 24 интерактивных задания · 13 слайдов теории · 4 живых демо**
- Геймификация: XP, уровни, серии, маскот, бейдж, звук (офлайн, Web Audio)
- **Режим учителя:** кнопка 🔑, пароль `neuro2026` — встроенная памятка на каждом
  экране (видна только учителю), методичка с ключами ответов и кнопками «открыть»
  любого задания.

## Файлы
- `index.html` — каркас
- `styles.css` — тема «Нейро-Лаб»
- `app.js` — движок, геймификация, режим учителя
- `data.js` — весь контент урока + методичка
- `illustrations.js` — SVG-иллюстрации слайдов
- `demos.js` — интерактивные демо
- `sound.js` — звук
- `AI_kurs_plan.md` — план всего курса (7 уроков)

Пароль учителя меняется в первой строке `app.js`.
